#### 使用方法
 * 直接在需要用到的组件中添加以下内容即可：
  ```js
    <Tabs>
      <Tab title="title1" index='1'>内容1</Tab>
      <Tab title="title2" index='2'>内容2</Tab>
      <Tab title="title3" index='3'>内容3</Tab>
      ...
    </Tabs>
  ```
  label: tab切换的头部标题，必须有，
  index：用于判断标题的突出显示效果，比如高亮，必须有
  * 需要用`npm install`命令安装依赖文件才可查看效果
